QR코드 만드는 방법
  • 선을 자유롭게 그릴 수 있고 방법은 아래와 같습니다.
    1. 선 색상을 선택합니다.
    2. 입력창에 입력을 해서 선 두께를 설정합니다.
    3. 선 스타일을 선택합니다.
    4. 템플릿 위에 선을 자유롭게 그립니다.
  • 도형을 추가할 수 있고 방법은 아래와 같습니다.
    1. 도형의 안쪽 색상과 선 색상을 선택합니다.
    2. 입력창에 입력을 해서 도형의 선 두께를 설정합니다.
    3. 도형 모양을 선택합니다.
    4. 템플릿 위에 드래그해서 도형을 추가합니다.
  • 아이콘을 추가할 수 있고 방법은 아래와 같습니다.
    1. 아이콘 색상을 선택합니다.
    2. 아이콘 모양을 선택합니다.
    3. 템플릿 위에 드래그해서 아이콘을 추가합니다.
  • 텍스트를 추가할 수 있고 방법은 아래와 같습니다.
    1. 텍스트 굵기, 기울기, 밑줄여부를 선택합니다.
    2. 텍스트 색상을 선택합니다.
    3. 입력창에 입력을 해서 텍스트 크기를 설정합니다.
    4. 텍스트 스타일을 선택합니다.
    5. 템플릿 위에 클릭해서 텍스트를 추가한 후에 텍스트 내용을 수정합니다.
    6. 텍스트를 선택한 후에 정렬을 선택합니다.
  • 템플릿을 변경할 수 있고 방법은 아래와 같습니다.
    1. 변경할 템플릿을 선택합니다.
    단, 작업하던 중에 템플릿을 변경하면 이전에 작업한 내역들은 모두 초기화되니 이 점은 유의해주시기 바랍니다.
  • 사진을 추가할 수 있고 방법은 아래와 같습니다.
    1. 사진 선택창이 노출됩니다.
    2. 사진 선택창에서 추가할 사진을 선택하면 템플릿 위에 사진이 추가됩니다.
  • 선택한 항목을 삭제할 수 있고 방법은 아래와 같습니다.
    1. 삭제할 항목을 선택합니다.
    참고로 항목을 선택했을 때 보이는 점들 중에서 아래 점 9개는 항목의 크기를 변경할 수 있고 위에 점 1개는 항목을 회전시킬 수 있습니다.
    2. 삭제를 누르면 항목이 삭제됩니다.
    3. QR코드와 안심번호는 반드시 필요한 항목이므로 삭제하실 수 없습니다.
